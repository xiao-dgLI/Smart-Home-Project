如果没有代理软件，可以用 SSH 方式代替 HTTPS，成功率更高：


bash
# 1. 先生成 SSH 密钥（如果之前没生成过）
ssh-keygen -t rsa -b 4096 -C "你的邮箱@example.com"
# 出现提示直接按 3 次回车

# 2. 查看公钥
type %USERPROFILE%\.ssh\id_rsa.pub
# 复制输出的全部内容

通过 Git CMD 操作（推荐）

1. 克隆仓库到本地

bash
cd C:\Users\你的用户名\Desktop
git clone https://github.com/xiao-dgLI/Smart-Home-Project.git
cd C:\Users\你的用户名\Desktop
git clone https://github.com/xiao-dgLI/Smart-Home-Project.git

2. 把你的文件复制到 android app 文件夹

在文件资源管理器中打开：

text
C:\Users\你的用户名\Desktop\Smart-Home-Project\android app\
C:\Users\你的用户名\Desktop\Smart-Home-Project\android app\

将你要上传的所有文件直接复制粘贴到这个文件夹中。


3. 提交并推送

bash
cd C:\Users\你的用户名\Desktop\Smart-Home-Project

git status

git add "android app/"

git commit -m "上传文件到 android app 文件夹"

git push origin main
cd C:\Users\你的用户名\Desktop\Smart-Home-Project

git status

git add "android app/"

git commit -m "上传文件到 android app 文件夹"

git push origin main

注意：因为文件夹名含有空格，路径必须用双引号包裹。




后续如果需要再次上传文件，只需要重复以下步骤：
bash
git add .
git commit -m "说明你做了什么修改"
git push origin main
git add .
git commit -m "说明你做了什么修改"
git push origin main
